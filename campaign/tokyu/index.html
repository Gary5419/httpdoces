キャンペーン詳細
TOKYU POINT 5％加算条件
キャンペーン対象期間中に、ポケカルに掲載の商品（ツアーや散策・体験イベント、チケット）をTOKYU CARDでご決済され、実際に商品をご利用された方（ツアー・イベントへの参加）にもれなく、5％分のTOKYU POINT※を加算いたします。 ※通常ポイント1％＋本キャンペーン分の4％
ご利用対象期間
2018年2月20日(火)～5月15日(火)までにTOKYU CARDで決済された方が対象となります。
ポイント加算の対象外条件
以下はポイント加算の対象外となります。
・予約後のキャンセル
・ツアー・イベントへの不参加
・現地決済
注意事項
※加算されたポイントはTOKYU POINT Webサービス 「TOKYUポイント履歴参照」にてご確認いただけます。
※キャンセルの場合はキャンペーン対象外となります。
※内容は予告なく変更になる場合がございます。あらかじめご了承ください。
※【TOP】のマークの付いたTOKYU CARDが対象です。JALカードTOKYU POINT ClubQ、ANA TOKYU POINT ClubQ PASMOマスターカード、スーパーICカード TOKYU POINT PASMO「三菱東京UFJ-VISA」は5%加算の対象外となります。
※TOKYU POINTは、通常の1％分は、「毎月」加算となり、本キャンペーンの４％分は、2018年7月下旬までに加算いたします。
